Дополнительные Опции

--- Добавляет вертикальную лигию (Разделитель)

-img- Url Картинки -#- Добавляет картинку (Пример: -img- unsplash.com/image/image_id -#- )

-a- Ссылка -enda- Добавляет Ссылку (Пример: -a- https://google.com -enda- )

-h3- Текст -endh3- Добавляет Заголовок (Пример: -р3- Привет мир -endh3- )

-v- Уникальная ссылка -endv- Добавляет Видео с YouTube (Пример: -v- hFrt23$@Rs2 -endv- )
